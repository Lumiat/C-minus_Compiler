## 理论与实际工程的对应

| 理论步骤         | 对应的工程文件 (TINY 为例) | 你的实际操作内容                                                |
| ---------------- | -------------------------- | --------------------------------------------------------------- |
| 词法规则 (Regex) | TINY.L 或 SCAN.C           | 定义如何识别 ==, !=, int, while                                 |
| 语法规则 (BNF)   | TINY.Y 或 PARSE.C          | 描述 if-else 的嵌套结构，实现函数调用规则                       |
| 分析树生成       | UTIL.C (newTreeNode)       | 编写创建 AST 节点的函数，把识别出的信息填入结构体               |
| 语义/符号表      | SYMTAB.C, ANALYZE.C        | 遍历 AST。遇到 int x 存入符号表，遇到 x = 5 去表里查 x 是否定义 |
| 目标代码生成     | CGEN.C, CODE.C             | 再次遍历 AST。根据节点类型打印出虚拟机的汇编指令                |
